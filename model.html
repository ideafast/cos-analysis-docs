Model upload requirements

To upload a model, call the upload_model function with the path to the model directory.

This directory must meet the following requirements:

  • Required files: Ensure that both config.json and inference.py are present.

  • Model file: The directory must contain a model file named model.

  • Valid JSON: Check if the JSON file is correct and contains the right arguments (see JSON Format).

  • Inference Function: The inference.py file must define a run_inference function.

  • Output of the run_inference function: The run_inference function must output a pandas.DataFrame with the following columns : [‘subject_id’, ‘tup’, ‘tup_day’, ‘Date’, ‘TIMING’] and a prediction column named “predicted_{target_name}”.

  • run_inference function purpose: The run_inference function must execute model inferences on a set of subjects. This function loads a pre-trained model, performs necessary transformations on the dataset, applies the model, and returns predictions in a DataFrame format.

  • Function Parameters: Verify that run_inference has the following parameters : “subjs”, “version”, “directory_path” and “savepath” (optional).

  • No Unrelated Files: The directory must not contain any image or CSV files.

Before uploading, your model will be tested on a patient to ensure it runs correctly.

inference.py

 1import pandas as pd
 2import pickle
 3import json
 4
 5from cosanalysis import build_dataset_on_timings, get_timings
 6        
 7def run_inference(subjs, 
 8                  version,
 9                  directory_path,
10                  savepath=None):
11    """
12    Run model inferences on each tups associated to a list of subjects.
13    
14    Parameters:
15    - subjs: list of subject_id
16    - version
17    - directory_path
18    - savepath to save predictions (optional)
19    
20    Returns:
21    - pd.DataFrame: A dataframe with the predictions.
22    contains columns "subject_id", "tup", "tup_day", "Date", "TIMING", prediction
23    """
24    
25    config = load_json_file(f'{directory_path}/config.json')
26    
27    timings = get_timings(subjs, 
28                          config["dataset_metadata"]["window_size"], 
29                          version)
30    
31    # Load and prepare data
32    df_analysis = build_dataset_on_timings(device_types=config["dataset_metadata"]["device_types"], 
33                                           timings=timings, 
34                                           version=version,
35                                           imputation=config["dataset_metadata"]["imputation"]
36                                           )[["subject_id", "tup", 
37                                              "tup_day", "start_window", 
38                                              "TIMING"] + config["model_metadata"]["input_features"]]
39
40    df_analysis["Date"] = pd.to_datetime(df_analysis.start_window).dt.date
41    prediction_df = df_analysis[["subject_id", "tup", "tup_day", "Date", "TIMING"]].copy()
42
43    df_analysis = df_analysis[config["model_metadata"]["input_features"]]
44    
45    for col in df_analysis.columns:
46        df_analysis[col] = pd.to_numeric(df_analysis[col])
47            
48    # Load model
49    with open(f"{directory_path}/model.pkl", "rb") as file:
50        model = pickle.load(file)
51        
52    # Run inferences
53    predictions = model.predict(df_analysis)
54
55    # output
56    prediction_df[f"predicted_{config['prediction']['prediction_label']}"] = predictions
57
58    if savepath is not None:
59        prediction_df.to_csv(savepath, index=False)
60    
61    return(prediction_df)
62
63def load_json_file(filepath):
64    with open(filepath, "r") as file:
65        data = json.load(file)
66    for key, value in data.items():
67        if value == "None":
68            data[key] = None
69    return data
